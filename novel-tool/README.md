# 小说工具使用说明
本文件夹内含两款小说工具，无需修改代码，直接打开即可使用，所有内容左对齐。

## 小说创作器（write.html）
**核心用途**：撰写小说、整理小说设定、分章节编辑内容。
操作步骤：填写左侧小说基础设定，在右侧编辑章节标题与正文；点击**保存项目**，可下载带创作内容的HTML文件，本地双击即可离线续写；点击**导出TXT**，获取可导入生成器的小说文本。

## 小说生成器（tool.html）
**核心用途**：将TXT小说，制作成可直接阅读的网页版小说。
操作步骤：上传小说封面、作者头像，填写作品相关信息；导入创作器导出的TXT文件，点击**更新预览**查看效果，确认后点击**导出整本书**，解压导出文件即可使用。

## 配套使用流程
1. 打开**小说创作器**完成小说撰写，导出TXT文件
2. 打开**小说生成器**导入TXT，生成网页版小说

## 重要提醒
1. 两个工具文件名**切勿修改**，避免无法正常使用
2. 保存项目下载的HTML文件，仅用于本地备份续写
3. 两款工具相互独立，互不干扰、不会冲突
