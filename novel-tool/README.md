## 小说工具使用说明
本文件夹包含两款小说工具，无需修改代码，直接打开即可使用。

### 小说创作器（write.html）
**用途**：撰写小说、整理设定、分章节编辑。
操作：填写小说设定，编辑章节；点击**保存项目**可下载带内容的HTML，本地双击即可打开续写；点击**导出TXT**，获取小说文本。

### 小说生成器（tool.html）
**用途**：将TXT小说生成网页版阅读文件。
操作：上传封面、填写作品信息，导入TXT；点击**更新预览**查看效果，确认后**导出整本书**，解压即可使用。

### 使用流程
1. 打开**小说创作器**写小说，导出TXT
2. 打开**小说生成器**导入TXT，生成网页小说

### 注意事项
- 文件名**切勿修改**，避免无法访问
- 保存项目下载的HTML仅用于本地备份
- 两款工具相互独立，互不冲突
