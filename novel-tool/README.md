# 小说创作&生成工具说明
本文档为`novel-tool`文件夹内两款工具使用说明，内容简洁清晰，方便直接复制使用。

## 文件夹内含工具
1. **小说创作器（write.html）**
2. **小说生成器（tool.html）**

## 小说创作器（write.html）
### 工具定位
专属小说原创撰写工具，用于从零开始创作小说、整理小说设定，是前期内容创作核心工具。

### 核心功能
- 内置小说知识库，可填写**小说名称、世界观、人物设定、故事大纲、伏笔线索、创作灵感**六大创作模块，完善小说基础设定
- 支持分章节撰写正文，自动生成章节列表，可随时切换编辑已有章节
- 实时统计**当前章节字数、全书总字数**，清晰掌握创作进度
- 支持**明暗护眼模式**切换，长时间创作更舒适
- 可一键保存创作项目，关闭页面后再次打开能加载原有内容，不丢失创作数据
- 支持导出纯TXT正文文件，无缝对接小说生成器使用

### 使用步骤
1. 打开工具后，先在左侧知识库完善小说各项设定
2. 在右侧编辑区输入章节标题与正文内容，点击【保存章节】
3. 创作中途/完成后，点击【保存项目】留存所有数据
4. 创作完毕，点击【导出TXT】获取小说正文文件

## 小说生成器（tool.html）
### 工具定位
将创作好的小说，转化为带精美界面、可直接阅读的网页版小说，实现小说可视化分享。

### 核心功能
- 支持上传**小说封面、作者头像**，自定义小说标签、作品简介、作者信息
- 导入小说创作器导出的TXT文件，**自动识别章节、拆分章节内容**
- 实时预览小说完整主页效果，包含小说信息、作者展示、章节目录
- 一键打包导出ZIP文件，生成带目录、分章节、可跳转的网页版小说
- 生成的阅读页面自带**夜间模式、上下章跳转、返回目录**功能，阅读体验完善

### 使用步骤
1. 进入工具后，依次填写小说名称、标签、简介、作者相关信息
2. 上传小说封面、作者头像，导入创作好的TXT小说文件
3. 点击【更新预览】查看效果，确认无误后点击【导出整本书】
4. 解压导出的ZIP包，打开`index.html`即可查看完整网页版小说

## 双工具配合使用流程
1. 用**小说创作器（write.html）**完成小说撰写、设定整理，导出TXT文件
2. 用**小说生成器（tool.html）**导入TXT文件，完善小说展示信息
3. 生成网页版小说，完成创作到可视化展示全流程

## 注意事项
1. 两款工具均为静态HTML文件，**无需安装、无需修改代码**，直接打开即可使用
2. 文件名禁止修改，保持`write.html`、`tool.html`原名，避免访问异常
3. 工具存放于`novel-tool`文件夹内，无需更新代码，永久可用
